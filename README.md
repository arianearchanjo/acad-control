# 📚 AcadControl

SaaS de Gestão de Estudos

Projeto desenvolvido para a disciplina **Prática Profissional em
Programação Web**.

------------------------------------------------------------------------

## 🎯 Sobre o Projeto

O **AcadControl** é um SaaS de Gestão de Estudos que permite que
estudantes organizem suas matérias, acompanhem suas notas, registrem
horas estudadas e visualizem seu desempenho por meio de um dashboard
interativo.

O objetivo do sistema é oferecer controle, estratégia e visão clara do
desempenho acadêmico, auxiliando o aluno na tomada de decisões para
melhorar seus resultados.

------------------------------------------------------------------------

## 🚀 Funcionalidades Principais

-   Cadastro e login de usuário\
-   Cadastro de matérias\
-   Registro de avaliações e notas\
-   Cálculo automático de média ponderada\
-   Registro de horas estudadas\
-   Cálculo de meta de horas atingida\
-   Dashboard com indicadores de desempenho

## ⚙️ Como Executar o Projeto

### 1️⃣ Clonar o Repositório

``` bash
git clone <url-do-repositorio>
```

------------------------------------------------------------------------

### 2️⃣ Instalar Dependências (Backend)

Dentro da pasta `backend`:

``` bash
npm install
```

------------------------------------------------------------------------

### 3️⃣ Iniciar o Servidor

``` bash
node server.js
```

O servidor será iniciado em:

http://localhost:3000
